## 📝 **Project Description**

This Mini-EDA Project includes:

### ✔ Data Cleaning

* Handling missing values
* Fixing incorrect data types
* Cleaning text columns (strip, remove special characters)
* Extracting numeric values from rating column
* Creating helper features using value_counts()

### ✔ Exploratory Data Analysis

* Summary statistics (`describe()`)
* Distribution analysis (histograms, boxplots)
* Relationship analysis (scatterplots, pairplots)
* Category distribution (countplots, bar charts)
* Correlation heatmap

### ✔ Dataset Used

* Zomato Restaurants Dataset
  (Includes columns like Restaurant Name, City, Cuisines, Rating, Online Delivery, Votes, etc.)
